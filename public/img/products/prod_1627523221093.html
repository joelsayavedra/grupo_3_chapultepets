Retrospectivas

Primer sprint - Wireframes

Comenzar a hacer:

  • Reuniones más seguido.

Hacer más

  • Contemplar todas las versiones del sitio (para diferentes dispositivos).

Continuar haciendo

  • Repartir las tareas.

Hacer menos

  • Postergar las reuniones de trabajo.

Dejar de hacer

  • No contemplar la fecha de entrega.

Segundo sprint

Comenzar a hacer:

  • Definir las tareas para 2 personas.
  • Planificar el Sprint, poniendo tareas objetivas y medibles (SMART).

Hacer más

  • Reuniones daily. (Al menos 3 días a la semana)

Continuar haciendo

  • Plasmar las tareas en un tablero de trello.

Hacer menos

  • Videoconferencias para trabajar (No para organizarse)

Dejar de hacer

 

Tercer sprint

Comenzar a hacer:

  • Cronometrar adecuadamente los tiempos del daily para forzarnos a hablar más.
  • Revisar cada una de las rutas para pulirlas gráficamente el maquetado.
  • Escribir la retroalimentación en una sesión de videoconferencia.
  • Contemplar la complejidad de las tareas para puntuarlas.

Hacer más

  • Push en Git, para mantener actualizado el proyecto todo el tiempo.

Continuar haciendo

  • Reuniones tres días a la semana.
  • Planificar las tareas para dos personas.

Hacer menos

  • Dejar de planificar tareas en cada daily, tratar de tenerlas todas desde el principio.

Dejar de hacer

  • Postergar la decoración del sitio (agregar paleta de colores, fuentes e imágenes decorativas).

Cuarto sprint

Comenzar a hacer:

  • Revisar cada una de las rutas para pulirlas gráficamente el maquetado.

Hacer más

  • Revisar y actualizar el tablero de trello cada vez que se comienza y se termina, respectivamente, una tarea programada.
  • Proponer fechas límites para las actividades.

Continuar haciendo

  • Cronometrar adecuadamente los tiempos del daily para forzarnos a hablar más.
  • Hacer push cada vez que se termina una tarea.
  • Reuniones tres días a la semana.

Hacer menos

  • Ocasionar conflictos de cambios entrantes/actuales en el mismo archivo.

Dejar de hacer

  • Postergar la decoración del sitio (agregar paleta de colores, fuentes e imágenes decorativas).
  • Equivocarse en las tareas que le corresponden a cada quién (El Saya todo meco)
